# Permutaciones con repetición

Nos sirve para saber de cuántas formas diferentes se puede ordenar un [[Conjunto]] de $n$ elementos, pudiéndose utilizar el mismo elemento múltiples veces.

$$
P_{rep}^{(n, k)} = n^k
$$
