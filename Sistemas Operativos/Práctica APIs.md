# Práctica APIs

  

El método que que permite realizar una operación que crea un recurso en el servidor es el método **POST**, en el endpoint `http://localhost:80/post`.

  

![Request realizado y response obtenido](request-response.png)

  

Como el response tiene un estado HTTP `200`, la operación fué exitosa, y el recurso fué creado.

  

OpenWeather One Call API 3.0 presenta un modelo de negocio *pay-as-you-go*, ofreciendo una solución que integra datos meteorológicos en tiempo real, pronósticos avanzados y un archivo histórico de 47 años. La documentación es lo suficientemente clara para una integración inmediata, ya que proporciona una estructura técnica precisa con parámetros detallados, ejemplos de respuestas JSON para el modelado de datos y una guía de manejo de errores.
