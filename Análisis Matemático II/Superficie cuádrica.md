# Superficie cuádrica

$$
\left ( Ax^2 + By^2 + Cz^2 + D = 0 \right)
\lor
\left ( Ax^2 + By^2 + Cz = 0 \right)
$$

> [!important]
> Son simétricas respecto de los planos coordenados, pero no respecto del origen.
