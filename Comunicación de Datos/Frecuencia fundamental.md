# Frecuencia fundamental

Cuando todas las componentes de una [[Señal]] tienen [[Frecuencia|Frecuencias]] $n \cdot f$ múltiplo de una [[Frecuencia]] dada $f$, esta se denomina [[Frecuencia fundamental]].
