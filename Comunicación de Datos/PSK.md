# PSK

Técnica de [[Modulación]], dónde los [[Dato digital|Datos digitales]] se representan mediante múltiples [[Fase|Fases]] diferentes de una [[Señal]] portadora.

Existen múltiples implementaciones de la misma:

- [[BPSK]]
- [[DPSK]]
- [[QPSK]]
- [[OQPSK]]
- [[MPSK]]
