# Full-duplex

Un [[Medio]] o [[Transmisión]] bidireccional, donde ambas estaciones pueden transmitir [[Señal|Señales]] ==simultáneamente==.

> [!note]
> Para esto, el [[Medio]] transmite [[Señal|Señales]] en ambas sentidos al mismo tiempo.
