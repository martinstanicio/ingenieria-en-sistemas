# Operadores lógicos

Estos son los operadores que podemos utilizar en un [[Python]] para comparar dos valores numéricos.

- Mayor: `>`
- Mayor o igual: `>=`
- Menor: `<`
- Menor o igual: `<=`
- Igual: `==`
- Distinto: `!=`
