# Ley asociativa

Para que se cumpla, los elementos $a$, $b$ y $c$ de un [[Conjunto]] $A$, se deben poder agrupar de todas las formas posibles sin alterar el resultado de una [[OperaciÃ³n]] $*$.

$$
(a * b) * c = a * (b * c)
$$

> [!important]
> La [[Ley asociativa]] siempre se cumple para el [[Elemento neutro]] y el [[Elemento absorvente]].
