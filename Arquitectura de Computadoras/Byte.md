# Byte

Unidad que representa 8 [[Bit|bits]] en el [[Sistema binario]].

Por ejemplo, el número 5 escrito como un número binario de 8 bits.

| **MSB** |     |     |     |     |     |     | **LSB** |
| ------- | --- | --- | --- | --- | --- | --- | ------- |
| 0       | 0   | 0   | 0   | 0   | 1   | 0   | 1       |
