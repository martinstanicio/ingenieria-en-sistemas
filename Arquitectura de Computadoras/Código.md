# Código

Una representación **digital** de una **señal analógica**.

Existen dos tipos de [[Código|Códigos]]:

- **Alfanumericos** (por ejemplo, *ASCII*)
- **Numéricos**
	- [[Código libre]]
	- [[Código ponderado]] (por ejemplo, *8-4-2-1* y *2-4-2-1*)
