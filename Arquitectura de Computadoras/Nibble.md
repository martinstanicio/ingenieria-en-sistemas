# Nibble

Unidad que representa 4 [[Bit|Bits]] en el [[Sistema binario]].

Por ejemplo, el número $5$ escrito como un número binario de 4 [[Bit|Bits]].

| **MSB** |     |     | **LSB** |
| ------- | --- | --- | ------- |
| 0       | 1   | 0   | 1       |
